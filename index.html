Integración de Formulario API

Infraestructura backend gratuita y robusta para desarrolladores. Envía datos de formularios directamente a tu email con soporte para archivos adjuntos. Desarrollado por Joshua Chávez.

Ver Documentación Live Demo

Endpoint Principal

POST Method

Envía tus peticiones a esta ruta utilizando cualquier tecnología que soporte HTTP POST (HTML, React, Vue, Node.js, etc).

https://forms-nicpages.vercel.app/submit-form

Variables de Configuración

to Obligatorio

El correo electrónico donde recibirás los datos del formulario.

redirect_to Opcional

URL absoluta (ej: https://tusitio.com/gracias) para redirigir al usuario tras el éxito.

files Opcional

Input tipo 'file' para adjuntar documentos (.pdf, .doc, imágenes). Soporta múltiples archivos.

Campos Libres Dinámico

name, phone, city, message... Cualquier campo extra será incluido en el cuerpo del correo.

HTML5 Nativo

Integración directa sin JavaScript. Ideal para landing pages estáticas.

index.html
<form action="..." method="POST" enctype="multipart/form-data">
  <input type="hidden" name="to" value="tu@email.com">
  <input type="text" name="nombre">
  <button type="submit">Enviar</button>
</form>

React / Next.js

Ejemplo asíncrono utilizando Fetch API y FormData.

ContactForm.jsx
const handleSubmit = async (e) => {
  e.preventDefault();
  const formData = new FormData(e.target);
  formData.append('to', 'tu@email.com');

  await fetch('https://...', {
    method: 'POST',
    body: formData
  });
};
