Segura & Anónima

API Backend desarrollada con Python & FastAPI. Procesa datos de manera encriptada y los entrega vía SMTP seguro a través de subdominios verificados.

Core: Python + FastAPI

Arquitectura asíncrona de alto rendimiento. Validación estricta de tipos con Pydantic para asegurar la integridad de los datos.

Motor: Resend

Integración directa con la API de Resend. Garantiza una entregabilidad del 99.9% evitando la carpeta de SPAM.

Privacidad & Anonimato

Tus datos viajan encriptados. El sistema actúa como un "túnel ciego": recibe, procesa y olvida. No almacenamos logs.

Enrutamiento Verificado

Uso de subdominio dedicado en Resend. Autenticación DKIM/SPF para firmar digitalmente cada envío.

Guía de Integración

POST https://forms-nicpages.vercel.app/submit-form
Parámetros (FormData)
  • to Requerido
    Email destino. Debe ser un correo válido.
  • redirect_to Opcional
    URL absoluta para redirigir al usuario tras el envío.
  • files Binario
    Soporte para adjuntos (imágenes, pdf, doc).

Ejemplo React + Vite

Recomendado
src/components/ContactForm.jsx 
import React, { useState } from 'react';
import { motion } from 'framer-motion';

const Contact = () => {
  const [formData, setFormData] = useState({
    name: '', email: '', message: '', file: null,
    // Configuración del API
    to: 'joshua44benja@gmail.com',  // Cambia este correo por el tuyo
    redirect_to: 'https://misitio.com/gracias' 
  });

  const handleSubmit = async (e) => {
    e.preventDefault();
    
    const payload = new FormData();
    // Append dinámico de datos
    Object.keys(formData).forEach(key => {
       payload.append(key, formData[key]);
    });

    // Envío seguro a través de la API Python/FastAPI
    await fetch('https://forms-nicpages.vercel.app/submit-form', {
      method: 'POST',
      body: payload,
    });
    
    alert('¡Enviado y encriptado!');
  };

  return (
    <form onSubmit={handleSubmit}>
      {/* Tus inputs aquí */}
    </form>
  );
};

Este código utiliza FormData nativo para manejar archivos binarios correctamente.
